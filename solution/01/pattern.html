PHP is a preprocessor that converts PHP Scripts into HTML. A valid PHP file should produce valid HTML output.

Problem

Before we can build applications we need a reliable place to serve them from. Select a web hosting environment that has the following criteria.

  • Spend less than $5/month
  • Choose a mainstream vendor that will be around in five years.
  • Domain registration bundled in
  • Control panel access to configure application
  • One click setup of WordPress
  • Supports PHP/MySQL with no additional setup

Solution

Search the internet for most popular web hosting platforms. Select a host that meets all of the listed criteria.

Serving Files from Bluehost

A Bluehost server will process PHP files or HTML files. The suffix dictates whether the code is converted to HTML.

Create a file "index.php" with the following content "< h1> My Bluehost Home Page < / h1>"

Use Bluehost "File Manager" to move a simple file (index.php) to Bluehost.

Review the welcome email. Note the "Domain" setting. This will be the URL of your website that you will type into the browser. Visit your domain in the browser.
